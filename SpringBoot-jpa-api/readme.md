## Hospital
리뷰 등록 및 조회 기능

### Endpoint
#### Review 등록
`/api/v1/hospitals/{id}/reviews`

#### 해당 id 병원의 Review 조회
`/api/v1/hospitals/{id}/reviews`

#### 병원 정보와 해당 병원의 Review 조회
`/api/v1/hospitals/{id}`

#### Review 1개 조회
`/api/v1/reviews/{id}`
